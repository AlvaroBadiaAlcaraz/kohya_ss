Requisitos del sistema:

Aunque los modelos LoRA sean los menos exigentes para entrenar un modelo de generación de imágenes estos siguen necesitando un mínimo de recursos.
Para asegurar que los resultados sean correctos y no tarde el proceso demasiado tiempo es recomendable que el ordenador en el que se ejecute tenga al menos 8 GB de VRAM

Clave de Clarifai:

Para poder hacer uso de esta interfaz es necesario contar con una cuenta de la página web Clarifai, ya que ciertas partes del proceso hacen uso de métodos proporcionados por APIs de esta página.
Enlace a la página de Clarifai

Una vez creada la cuenta, en la pestaña de "Account" (tu cuenta), en el apartado de seguridad "Security", podrás generar un token personal, este token es el necesario para poder hacer uso de la interfaz.
