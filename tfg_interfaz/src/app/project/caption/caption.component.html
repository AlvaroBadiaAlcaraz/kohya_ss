Paso 2: Corrección de descripciones de las imágenes

A continuación se mostrarán las imágenes que ha introducido al modelo para entrenarlo, de manera automática se ha escrito una pequeña descripción de cada una de ellas.
En el caso de que considere que una de estas descripciones no es lo suficientemente buena o no se adecua correctamente a lo que se muestra en la imagen puede modificarla, para que los cambios sean efectivos tras modificar la descripción deberá hacer click en el botón de "Guardar cambios"

Para que las descripciones puedan ser utilizadas en el entrenamiento del modelo deberñan comenzar todas con el nombre del sujeto que declaró con anterioridad.
Por ejemplo, si en el paso anterior introdujo como nombre del sujeto JohnDoe, las descripciones deberñan comenzar de la siguiente manera: JohnDoe is a man with long hair, blue eyes, etc.

Las descripciones deben estar en inglés, ya que es este el idioma que entiende el modelo para entrenar, y deberían centrarse en el sujeto sobre el que se esta entrenando, describiendo sus atributos físicos principalmente.
Procure ser consistente con sus descripciones, ya que el modelo usarña los atributos comunes entre todas las imagenes para realizar el entrenamiento.
Evitar añadir a las descripciones detalles del fondo, ya que no son relevante spara obtener un buen modelo y pueden confundir al modelo en el proceso de entrenamiento.

Las descripciones deberían contener entre 25 y 75 palabras, cuantos mas detalles se den en cada una de las imágenes, mejor será el resultado del modelo.
