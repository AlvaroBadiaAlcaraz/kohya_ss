Paso 1: Creación del entorno de entrenamiento

1. Introducir las imágenes de entrenamiento

Elija las imágenes con las que entrenará el modelo de LoRA que quiera generar.
Introduzca todas las imágenes que quiera usar a la vez.




2. Especificar el nombre del modelo y que clase de modelo es.

El nombre que elija en este paso será el que más adelante identifique a su modelo.
El nombre es recomendable que este compuesto por nombre y primer apellido escritos juntos. Por ejemplo: JohnDoe



La clase será utilizada por el modelo durante el entrenamiento.
En caso de estar entrenando el modelo de un hombre elija la opción de hombre y en caso de ser una mujer elija la opción de mujer
